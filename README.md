# Kunal Oberoi Portfolio

A personal portfolio web application built with React to showcase projects, technical skills, tools, and resume.


## Live Demo

https://kunaloberoiportfolio.vercel.app/

## Live Sections

- Home page with intro, animated effects, and social links
- About page with professional summary, skill stack, and tools
- Projects page with project cards and demo/source links
- Resume page with in-browser PDF preview and download option

## Tech Stack

- React 17
- React Router DOM 6
- React Bootstrap + Bootstrap 5
- React Icons
- React TSParticles
- React PDF

## Project Structure

- public: static app shell files
- src: app source code
- src/components: reusable UI sections (Home, About, Projects, Resume, Navbar, Footer)
- src/Assets: images, icons, and resume PDF

## Getting Started

Prerequisites:

- Node.js 18+ recommended
- npm

Install and run:

1. Clone the repository
2. Open the project folder
3. Install dependencies:
   npm install
4. Start development server:
   npm start

The app runs at http://localhost:3000

## Available Scripts

In the project directory, you can run:

- npm start: Runs the app in development mode
- npm run build: Builds optimized production files in the build folder
- npm test: Runs tests in watch mode
- npm run eject: Ejects Create React App config (one-way operation)

## Build and Deployment

Create a production build:

npm run build

Serve the production build locally:

npx serve -s build

## Notes

- This project uses Create React App tooling.
- Browserslist data should be updated periodically when prompted.
